O PROJETO:
A ideia da CRONOS é oferecer dois diferentes serviços para os desenvolvedores: uma calculadora capaz de fornecer o valor médio cobrado por hora por um dev, levando em consideração a linguagem utilizada, o tempo de experiência e a região do país onde ele se encontra, além de uma dashboard para armazenar e averiguar o tempo investido nos projetos. 

Vale lembrar que a dashboard dá a liberdade de o dev adicionar o nome do projeto, bem como o valor que deseja cobrar por hora. Com um cronômetro presente na mesma linha, o profissional pode dar "play" assim que começar o trabalho, onde o valor será atualizado segundo a segundo.
